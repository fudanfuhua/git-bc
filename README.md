# git-bc

[![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Code coverage][coveralls-image]][coveralls-url]

> Git plugin to interactively list branches and checkout


## Install

```
$ npm install --g git-bc
```


## Usage

```js
git bc
```
